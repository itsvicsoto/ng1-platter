## Prereqs

```
* Install latest nodejs (should work on the latest version) but 5.7.1 is being used atm
```

## How to run?


```
## At this point you should be at the project root folder after cloning

$ npm install
$ bower install

## Note : usually after running gulp you might encountered some dependency
## issues or packages that are required e.g. node-sass just install them as
## see fit

## build the app
$ gulp dashboard-dev

## build the website
$ gulp website-dev
$ node .

## it will live in localhost:3001 
```
